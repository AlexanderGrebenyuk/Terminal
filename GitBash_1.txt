Linux terminal (GitBash) commands

1) Посмотреть где я
pwd 
 
2) Создать папку
mkdir new-papka
 
3) Зайти в папку
cd new-papka
 
4) Создать 3 папки 
mkdir {1,2,3} // создаст папки с именами 1, 2 ,3
 
5) Зайти в любую папку
cd /тут пишется путь/

6) Создать 5 файлов (3 txt, 2 json)
touch file1.txt file2.txt file3.txt file4.json file5.json
 
7) Создать 3 папки
mkdir {folder1,folder2,folder3}
 
8. Вывести список содержимого папки
ls -la
 
9) + Открыть любой txt файл
cat file1.txt //посмотреть содержимое файла
 
10) + написать туда что-нибудь, любой текст.
echo "this is a line" > file1.txt
cat file1.txt
 
11) Выйти из папки на уровень выше
cd ..
 
12) переместить любые 2 файла, которые вы создали, в любую другую папку.
 mv file1.txt /указать путь куда
 
13) скопировать любые 2 файла, которые вы создали, в любую другую папку.
cp file1.txt file2.txt /указать путь куда
 
14) Найти файл по имени
find . -name "file1.txt"
 
15) вывести несколько первых строк из текстового файла
head -10 file1.txt //выведет первые 10 строк из файла
 
16) вывести несколько последних строк из текстового файла
tail -10 file1.txt //выведет последние 10 строк из файла
 
17) вывести дату и время
Date
